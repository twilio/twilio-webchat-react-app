To contribute to the `Twilio Webchat React App`. Look for issues with tag
"good first issue" (or any other label). If the issue is not claimed, ask for permission to work on the issue; once a team member agrees, issue will be assigned to your name. 

Create a fork repository, then a feature branch. Work on your changes in your local, and push it to your remote forked feature/issue branch. Then create a PR on the original repo 

[ https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/proposing-changes-to-your-work-with-pull-requests/creating-a-pull-request-from-a-fork ]

Once the PR gets approved, add the `Merge` label. This'll merge the PR into the repository.
